# GameFavorite
お気に入りのゲームソフトなどをレビュー形式で投稿し、共有することができるサイトです。  
他の人のレビューを見たり、ログインすることでいいね機能などが使用可能となり、いいね数に応じたランキングもあります。  
ランキングによって、現在どのようなゲームが流行っているのかなどがすぐにわかります。

URL: https://game-favorite.herokuapp.com/
